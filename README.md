# Example of A Simple Web Application

# What's this?

- Frontend: React (create-react-app)
- Backend: vibe.d
- Output: Docker image

# Usage

- For development
  - `rdmd start`
    - Auto restart server on save source
    - Hot reloading scripts on save source
- For build image
  - `rdmd build`
    - Build Dockerfile with release build

# Requirements

- Node.js
- Docker
